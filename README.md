# Pong — Godot Training Project

A simple remake of **Pong** made with **Godot** and **GDScript**.

This project was created as part of my game development learning journey.
The goal was to practice the basics of 2D gameplay: player movement, ball movement, collisions, scoring, win conditions, and game reset.

---

The playable version is available on itch.io:

## Controls

| Action            | Input                             |
| ----------------- | --------------------------------- |
| Move left player  | Z / S or assigned keys            |
| Move right player | Up / Down arrows or assigned keys |
| Restart game      | Space                             |

---

## Features

* Two-player Pong gameplay
* Paddle movement
* Ball movement
* Wall and paddle collisions
* Score system
* Win condition
* Victory message
* Game reset system

---

## What I learned

This project helped me practice:

* using Godot scenes and nodes;
* scripting with GDScript;
* moving objects in 2D;
* handling player input;
* detecting score zones;
* updating UI labels;
* creating win conditions;
* resetting a game state;
* exporting a Godot project.

# Pong — Projet d’apprentissage Godot

Un remake simple de **Pong** réalisé avec **Godot** et **GDScript**.

Ce projet a été créé dans le cadre de mon apprentissage du développement de jeux vidéo.
L’objectif était de travailler les bases du gameplay 2D : déplacement des joueurs, déplacement de la balle, collisions, score, condition de victoire et reset de partie.
